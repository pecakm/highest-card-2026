# Highest Card (2026)

A quick bluff-and-guess card game you can play with friends in the same room or online. Made with Next.js 16 and PartyKit.

Live demo [here](https://highest-card.mpecak.pl).

Copyright 2026 IT Pulse Mikołaj Pęcak. All rights reserved.

## Getting started

1. Create `.env.local` file and fill the data correctly
2. `npm ci` to install dependencies
3. `npm run partykit:dev` to run the PartyKit server locally.
4. `npm run dev` to start the app locally (http://localhost:3000 by default)

Enjoy!
